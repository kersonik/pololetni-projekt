# Computer Science Topics - Documentation

## Overview
This project contains a series of educational HTML files covering core concepts in computer science. The documentation is structured to provide a clear understanding of hardware, software, networks, artificial intelligence, computing history, and supercomputing. Each file follows a consistent template with navigation menus, headings, definitions, and references.

## File Structure & Contents
- **`index.html`** [4]: Main entry point defining programs and data, with a navigation menu linking to all other topics.
- **`historie.html`** [5]: Chronicles the evolution of computers across five generations, from vacuum tubes and transistors to integrated circuits and modern microprocessors.
- **`hardware.html`** [6]: Details essential physical components, including CPUs, GPUs, RAM, storage drives (HDD/SSD), and motherboards with chipset functionality.
- **`software.html`** [2]: Explains software classification (system vs. application), the relationship between software and data, and common malware types (viruses, spyware, trojans) with defense strategies.
- **`site.html`** [3]: Covers computer networks, including OSI/TCP-IP models, switching methods (circuit vs. packet), architectures (P2P vs. client-server), signal types, and network scales (PAN, LAN, MAN, WAN).
- **`ai.html`** [7]: Introduces artificial intelligence, machine learning, neural networks, NLP, successful applications in games and industry, and key challenges like the "black box" problem, bias, hallucinations, and the EU AI Act.
- **`super.html`** [1]: Describes supercomputing architectures (MIMD, SIMD, SMP, NUMA, GPGPU), distributed computing projects (SETI@home, BOINC, Folding@home), and special-purpose machines (Deep Blue, GRAPE, Deep Crack).

## Key Features
- Consistent navigation and footer across all files
- Educational focus with clear definitions and historical context
- References primarily to Wikipedia for further reading
- Structured for easy learning and quick reference
